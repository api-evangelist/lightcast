# Lightcast (lightcast)

Lightcast (formerly Emsi Burning Glass) provides labor market data and analytics including the Career Coach Careers API, a RESTful service that contains economic data for careers stored in JSON format.

## APIs

### Lightcast Careers API

The Career Coach Careers API is a RESTful API service that contains economic data for all careers in the Career Coach app stored in JSON format.
